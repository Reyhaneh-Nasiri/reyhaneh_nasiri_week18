## Setup
### 1.Installation
```npm install```

### 2.Run the Project
```npm run dev```

### 3.Run the API
- This command must be run in a new terminal.
- Without running JSON Server, the application will not receive any data.
- Port 3000 is reserved for API.

```json-server --watch db.json --port 3000```
